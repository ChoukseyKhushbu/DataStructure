Given n number of words, you need to find if there exist any two words which can be joined to make a palindrome or any word itself is a palindrome.
Return true or false.

Sample Input 1 :
4
abc def ghi cba

Sample Output 1 :
true

Sample Output 1 Expalanation :
"abc" and "cba" forms a palindrome

Sample Input 2 :
4
abc def ghi hg

Sample Output 2 :
true

Sample Output 2 Expalanation :
"ghi" and "hg" forms a palindrome
